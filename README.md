# Sistema de Inventário Interativo

Trabalho Discente Efetivo da disciplina de Programação Lógica e Funcional. Ministrada pelo Professor Frank de Alcântara na instituição Pontifícia Universidade Católica do Paraná - PUCPR, para o 4° semestre do curso Bacharelado em Ciência da Computação.

## Visão Geral do Projeto
Esse projeto implementa um sistema de inventário feito em Haskell, projetado para ser interativo com o usuário por meio do terminal, sem uma interface gráfica. Tem como objetivo praticar os conceitos de programação funcional, manipulação de estado e operações I/O, fazendo o registro de todas as operações em um log de auditoria e persistindo o estado atual no disco, garantindo que não aja perca de dados durante as execuções.
